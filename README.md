# blood-bank-management-system

## Features

- **Centralized Blood Inventory Management**  
  Tracks blood units across multiple hospitals and blood banks, including blood group, quantity, storage status, and expiry details.

- **Blood Compatibility Validation**  
  Automatically verifies donor–recipient blood group compatibility before processing transfusion requests.

- **Automated Inventory Updates**  
  Database triggers ensure real-time updates to blood stock whenever donations, transfers, or requests are processed.

- **Blood Request and Transfer System**  
  Stored procedures manage blood requests between hospitals and handle approval workflows for secure and controlled transfers.

- **Donor and Hospital Records Management**  
  Maintains structured records of donors, hospitals, and blood banks to support efficient coordination and tracking.

- **Analytical Reporting for Hospital Networks**  
  SQL queries generate insights such as blood stock availability, demand trends, and hospital-level usage statistics.
